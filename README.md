# 📖 Descripción

La productividad y los niveles de energía son factores fundamentales para el rendimiento personal y profesional. Sin embargo, diversos hábitos diarios pueden afectar significativamente el desempeño de las personas sin que estas sean conscientes de ello.

Este proyecto desarrolla un análisis de datos para identificar la relación entre el descanso, el estrés y el uso de dispositivos electrónicos con el nivel de productividad y energía de la población.

Mediante Python, Pandas y Power BI se transforman datos en información estratégica que permite comprender qué hábitos tienen mayor impacto sobre el rendimiento diario.

---

# 🎯 Problema de Negocio

Empresas y organizaciones buscan mejorar el bienestar y la productividad de sus colaboradores, pero muchas veces desconocen cuáles son los factores que afectan el desempeño.

Este proyecto responde preguntas clave:

- ¿Cómo influye la cantidad de horas de sueño en la productividad?
- ¿Qué impacto tiene el uso del celular antes de dormir?
- ¿Existe relación entre el estrés y los niveles de energía?
- ¿Las horas frente a pantallas afectan el rendimiento?
- ¿Qué hábitos presentan las personas con mayor productividad?

---

# 📈 Principales Hallazgos

El análisis permitió identificar patrones relevantes:

- Las personas con menos horas de sueño presentan menores niveles de energía.
- Un mayor uso del celular antes de dormir se relaciona con una disminución del rendimiento.
- Los niveles elevados de estrés afectan negativamente la productividad.
- El exceso de tiempo frente a pantallas muestra una relación con la fatiga y el bajo desempeño.
- Los mejores niveles de productividad se observan en personas con hábitos de descanso más saludables.
